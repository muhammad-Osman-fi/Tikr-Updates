Tikr - Event Organizer
======================

Thank you for downloading Tikr!

Tikr is a lightweight, portable event organizer and guest management app.  
It helps you manage invitations, track attendance, analyze statistics, and 
export records — all in one simple tool.

-------------------------------------------------
📦 Installation
-------------------------------------------------
1. Tikr is portable — no installation required.
2. Unzip this package to any folder on your computer.
3. Run the application:
   - On Windows: Double-click "Tikr.exe"
   - On Linux/macOS: Open a terminal and run "./Tikr"
4. On first launch, you may be asked to activate your license key.

-------------------------------------------------
🔑 License Activation
-------------------------------------------------
- A valid license key is required to use Tikr.
- Enter your license key when prompted. Tikr will validate it online.
- Licenses are device-bound and may expire depending on your subscription plan.

-------------------------------------------------
⚙️ Features
-------------------------------------------------
- 📋 Manage invited and present guest lists
- ➕ Add guests manually with full details (name, email, event, status, etc.)
- ✉️ Send invitations via email (SendGrid credentials required)
- 🧾 Generate and attach QR codes to invitations
- 📸 Scan QR codes with your webcam for quick guest check-ins
- 💳 Track payment status, event type, gender, and guest status
- 📊 Real-time dashboard with attendance and payment statistics
- 🔍 Search and filter guests by name, email, date, time, gender, status, and more
- 🧹 Clear guest data, templates, and credentials (or reset everything)
- 📁 Export guest data in CSV, JSON, or Excel format
- ⚙️ Built-in settings panel to configure preferences
- 📝 Logs actions and errors to `tikr.log` for troubleshooting

-------------------------------------------------
🖥️ System Requirements
-------------------------------------------------
- Windows (32-bit or 64-bit) OR Linux OR macOS
- Internet connection (required for license validation and email sending)
- Webcam (required for QR code scanning)
- At least 200 MB free disk space

-------------------------------------------------
📖 Usage
-------------------------------------------------
1. Add guests using the "Add Guest" form.
2. Configure your email credentials under "Settings" → "Invitations".
3. Send invitations with QR codes to selected guests.
4. Use "Scan QR" to verify guests on arrival.
5. View real-time stats and dashboard metrics.
6. Export guest records (CSV/JSON/Excel) from the "Export Data" menu.
7. Use "Clear Data" if you need to reset guests, templates, or credentials.

-------------------------------------------------
🚀 Updating Tikr
-------------------------------------------------
- Tikr automatically checks for updates.
- If an update is available, it will be downloaded and applied on next launch.
- Updates are distributed via GitHub Releases.
- You can also update manually by downloading the latest release from GitHub.

-------------------------------------------------
📂 Files Created
-------------------------------------------------
- tikr.log → Log file (actions, errors)
- /exports/ → Guest exports (timestamped CSV/JSON/Excel files)
